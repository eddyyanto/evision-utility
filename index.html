E-Vision 6800

A command line utility to switch on/off the E-Vision 6800 projector.

Usage

evision.exe ip address on|off

  • First parameter: IP address of the projector — make sure you can reach the projector by pinging it from the computer running the utility.
  • Second parameter: On or off value — case insensitive.

Example

To switch on projector 192.168.14.101

evision.exe 192.168.14.101 on

To switch off projector 192.167.14.101

evision.exe 192.168.14.101 off
